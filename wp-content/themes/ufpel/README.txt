== Descrição ==

UFPel 2016 é a versão totalmente personalizável do tema originalmente
desenvolvido para os websites institucionais da Universidade Federal de Pelotas.
Implementa um versátil editor de layout, que permite personalizar a página
inicial do site de forma visual, e conta com layout responsivo para excelente
adaptação a dispositivos móveis. Oferece opção de logo personalizado no
cabeçalho, duas áreas de menus, modelo de página interna sem barra lateral,
opção de alto contraste (acessibilidade) e widgets exclusivos. A barra de
identidade visual do Governo Federal pode ser exibida, opcionalmente, no topo
do site.
